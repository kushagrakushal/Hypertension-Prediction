Project Overview

Predictive Pulse is a machine learning–based web application for predicting hypertension (high blood pressure) risk.

The system analyzes patient health data such as age, symptoms, medical history, and blood pressure readings.

A Django web interface allows users to input data and view risk assessment results.

Workflow

Patient enters health data in the web application

Data is sent to the Django backend

The trained machine learning model processes the input

The model predicts the hypertension risk stage

Results are displayed with risk level and recommendations

Technologies Used

Django

Scikit-learn

Pandas & NumPy

HTML / CSS

Input Features

Gender

Age

Family history of hypertension

Medication status

Symptoms (breath shortness, visual changes, nose bleeding)

Systolic blood pressure

Diastolic blood pressure

Diet and lifestyle factors

Output

Hypertension stage prediction

Confidence score

Clinical recommendations
