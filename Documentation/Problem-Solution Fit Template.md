Project Title: Predictive Pulse: Harnessing ML for Blood Pressure Analysis
Team ID: Predictive Pulse Team

3. TRIGGERS (TR) — What triggers customers to act?
- A relative suffers a sudden hypertensive crisis
- Doctor flags elevated BP at a routine check-up
- News: hypertension is the #1 preventable killer
- Friend shares a health screening tool online
- Experiencing symptoms: headache, nosebleeds, shortness of breath, visual changes

10. YOUR SOLUTION (SL):
Predictive Pulse — Flask web app using Logistic Regression (95.2% accuracy, 100% Crisis
Recall) trained on 1,348 clinical records. Users input 13 health parameters; app outputs
hypertension stage (Normal / Stage-1 / Stage-2 / Crisis) with a confidence score and
personalised clinical recommendations per stage.

4. EMOTIONS: BEFORE / AFTER (EM):
  Before: Anxious → Uncertain → Avoidant → Fearful
  After (using Predictive Pulse): Informed → Confident → Proactive → Empowered
  Strategy: Reassuring clinical tone. Green for Normal, Amber/Red for risk stages.
